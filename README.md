# Customer-Churn-Prediction
This project predicts customer churn using machine learning techniques to identify customers likely to leave a business. It analyzes customer demographics, usage patterns, and service-related factors. The model helps businesses take proactive retention actions. The project demonstrates end-to-end ML workflow from data preprocessing to model.
# Customer Churn Prediction


Predict whether a customer will leave using historical data.

 Tools Used
- Python
- Pandas
- Scikit-learn

 Steps
- Data Cleaning
- Exploratory Data Analysis
- Model Training (Logistic Regression)

 Result
Built a basic ML model to predict customer churn.
